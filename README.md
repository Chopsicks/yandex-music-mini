# Яндекс.Музыка мини

Неофициальный компактный десктоп-клиент для Яндекс.Музыки с глобальными хоткеями, эквалайзером и персональной волной.

![Скриншот плеера](screenshot.png)  
*— Интерфейс приложения —*

## Особенности

- **Компактное окно** — всегда поверх других окон, не занимает много места
- **Глобальные горячие клавиши** — управляйте музыкой из любого приложения
- **Эквалайзер с пресетами** — 8 встроенных настроек (рок, поп, классика, джаз, басы и др.)
- **Моя волна** — персональная станция с настройкой настроения, языка и разнообразия
- **Любимые треки и плейлисты** — полная интеграция с вашей библиотекой
- **Поиск по трекам** — быстро находите и воспроизводите нужную музыку
- **Тёмная и светлая темы** — автоматическое переключение через настройки
- **Гибкие настройки** — качество звука, автозапуск, автовоспроизведение и многое другое

## Технологии

- Python 3.10+
- Flask — локальный веб-сервер интерфейса
- pywebview — отрисовка окна
- python-vlc — воспроизведение аудио
- yandex-music-api — работа с API Яндекс.Музыки
- keyboard — глобальные горячие клавиши

## Установка

### Windows

Скачайте последнюю версию установщика с [официального сайта](https://frizzylow.github.io/yandex-music-mini/) (или перейдите по ссылке ниже) и запустите его.

> **Системные требования:** Windows 10/11 (64-bit), 2 ГБ ОЗУ.

```bash
git clone https://github.com/frizzylow/yandex-music-mini.git
