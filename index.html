О проекте

Это неофициальная фанатская версия приложения для Яндекс.Музыки. Мы не связаны с Яндексом и не несём ответственности за использование программы.

Оригинальный сервис: Яндекс.Музыка.

Все права на контент принадлежат их владельцам. Приложение использует открытый API.

Технические детали

Программа написана на Python 3.10+ и использует следующие библиотеки:

  • Flask — лёгкий веб-сервер для интерфейса
  • pywebview — отрисовка окна на основе системного браузера
  • python-vlc — воспроизведение аудио
  • yandex-music — неофициальный API Яндекс.Музыки
  • keyboard — глобальные горячие клавиши
  • selenium — автоматическая авторизация (опционально)

Исходный код доступен на GitHub под лицензией MIT.

Для замены аватарки и иконки приложения поместите файлы avatar.png и icon.png в папку с сайтом. Размер: 200x200 px.

Ключевые возможности

Глобальные хоткеи

Управляйте музыкой из любого приложения: play/pause, следующий/предыдущий, лайк/дизлайк, громкость и скрытие окна. Полностью настраиваются.

Эквалайзер с пресетами

8 встроенных пресетов: рок, поп, классика, джаз, усиление басов и высоких. Выберите своё звучание.

Моя волна

Персональная станция с настройками настроения, языка и разнообразия. Умные рекомендации под ваш вкус.

Любимые треки и плейлисты

Добавляйте в избранное, управляйте плейлистами, создавайте радио на основе трека.
